# Template-Based Encounter System - Usage Guide

## Overview
The template-based encounter system allows healthcare providers to quickly create standardized encounter reports using pre-built templates. This ensures consistency, saves time, and improves documentation quality.

## How to Use Templates for Report Creation

### 1. Creating New Encounters from Templates

#### From Dashboard:
1. Go to the Dashboard
2. In the "Upcoming Appointments" section, click the **"New Encounter"** button
3. This will open the Template Selector

#### From Encounters List:
1. Navigate to **Encounters** → **Encounters List** in the sidebar
2. Click the **"New Encounter"** button at the top right
3. This will open the Template Selector

#### From Appointment Details:
1. Click on any appointment card from the Dashboard
2. In the appointment detail view, scroll to the "Encounter Detail" section
3. Click **"Create New Encounter"** button
4. This will open the Template Selector with patient data pre-filled

### 2. Template Selection Process

#### Step 1: Choose Template
- Browse available templates by category (General, Emergency, Follow-up, Pediatric, etc.)
- Use the search bar to find specific templates
- Filter by category using the dropdown
- Each template shows:
  - **Name & Description**: What the template is for
  - **Category**: Type of encounter (color-coded)
  - **Estimated Time**: How long it takes to complete
  - **Usage Count**: How many times it's been used
  - **Sections Preview**: What fields are included

#### Step 2: Template Information
- **Required vs Optional Fields**: Red asterisk (*) indicates required fields
- **Field Types**: Text areas, dropdowns, checkboxes, radio buttons, etc.
- **Progress Tracking**: See completion percentage as you fill out the form

### 3. Creating Encounter Reports

#### Form Completion:
1. **Patient Information**: Pre-filled from appointment/selection
2. **Template Sections**: Fill out each section based on the template
3. **Progress Bar**: Track your completion progress
4. **Validation**: Required fields must be completed before saving
5. **Save**: Click "Save Encounter" to create the report

#### Field Types Available:
- **Text Areas**: For detailed notes and observations
- **Dropdowns**: For standardized selections (e.g., triage levels)
- **Checkboxes**: For multiple symptom selections
- **Radio Buttons**: For single choice options (e.g., patient status)
- **Number Fields**: For measurements and counts
- **Date Fields**: For scheduling and timing

### 4. Template Management

#### Creating Custom Templates:
1. Go to **Encounters** → **Encounter Templates**
2. Click **"Create New Template"**
3. Use the Template Builder to:
   - Set template name and description
   - Choose category
   - Add sections with different field types
   - Mark fields as required or optional
   - Preview and test the template

#### Managing Existing Templates:
- **View**: See template details and structure
- **Edit**: Modify template sections and fields
- **Duplicate**: Create a copy to customize
- **Delete**: Remove unused templates

### 5. Benefits of Template-Based Reports

#### Consistency:
- Standardized format across all encounters
- Ensures all important information is captured
- Reduces documentation errors

#### Efficiency:
- Pre-built sections save time
- Quick form completion with guided fields
- Reusable templates for common encounter types

#### Quality:
- Structured data collection
- Required fields ensure completeness
- Professional, organized reports

### 6. Template Categories

#### General Consultation:
- Standard medical consultations
- Chief complaint, history, examination, assessment, treatment plan

#### Emergency Visit:
- Emergency department encounters
- Triage levels, vital signs, immediate care protocols

#### Follow-up Visit:
- Patient follow-up appointments
- Previous treatment review, current status, next steps

#### Pediatric Consultation:
- Children's medical visits
- Growth measurements, developmental milestones, vaccination status

### 7. Workflow Example

1. **Patient arrives** → Check appointment in Dashboard
2. **Click appointment card** → View appointment details
3. **Click "Create New Encounter"** → Opens template selector
4. **Choose appropriate template** → Based on visit type
5. **Fill out encounter form** → Using template structure
6. **Save encounter report** → Creates standardized documentation
7. **View in Encounters List** → Access completed reports

## Tips for Best Results

- **Choose the right template**: Select templates that match the encounter type
- **Complete required fields**: Ensure all mandatory information is provided
- **Use consistent terminology**: Follow template guidelines for standardization
- **Review before saving**: Check completeness and accuracy
- **Create custom templates**: Build templates for specialized encounter types

This system streamlines the encounter documentation process while maintaining high-quality, consistent medical records.
